Axis-Aligned Bounding Box. AABB consists of a position, a size, and several utility functions. It is typically used for fast overlap tests.

@:native("End")end:Vector3

Ending corner. This is calculated as godot.AABB.position plus godot.AABB.size. Setting this value will change the size.

Value: Getting is equivalent to value = godot.AABB.position + godot.AABB.size, setting is equivalent to godot.AABB.size = value - godot.AABB.position

@:native("Position")position:Vector3

Beginning corner. Typically has values lower than godot.AABB.end.

Value: Directly uses a private field.

@:native("Size")size:Vector3

Size from godot.AABB.position to godot.AABB.end. Typically all components are positive. If the size is negative, you can use godot.AABB.abs to fix it.

Value: Directly uses a private field.
